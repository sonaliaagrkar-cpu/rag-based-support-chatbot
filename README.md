Customer Support Assistant using RAG

This project is developed as a RAG internship project focused on building a customer support assistant using retrieval-based question answering.
The system takes information from a PDF knowledge base, converts it into embeddings, stores it in ChromaDB, and retrieves relevant context to answer user queries.

Key Functionalities
- PDF document ingestion
- Text chunking and vector embeddings
- Retrieval-Augmented Generation pipeline
- LangGraph workflow for query processing
- Human escalation support for unresolved queries

Technologies Used
Python, LangGraph, ChromaDB, LangChain, Sentence Transformers

Sample Support Queries
- How can I reset my password?
- What is the refund policy?
- How are complex queries escalated?

This project demonstrates RAG workflow design, routing logic, and HITL support implementation.
